# Лабораторная работа 6

## Добавление новых данных в БД с помощью Entity Framework

Выполнил: Кунёв Антон

### Описание работы

В этой лабораторной работе сделано Windows Forms приложение на C# для работы с базой данных `MotorbikeDB`.
Программа подключается к базе через Entity Framework, показывает список мотоциклов и позволяет добавлять и удалять записи.

### Что реализовано

- вывод данных из таблицы `Table_Motorbike` в `DataGridView`;
- вывод изображения мотоцикла при выборе строки;
- отдельная форма для добавления новой записи;
- выбор изображения через `OpenFileDialog`;
- сохранение выбранного изображения в папку `Pictures`;
- удаление выбранной записи из базы данных;
- удаление изображения, которое относится к удалённой записи;
- загрузка уникальных марок в `ComboBox`;
- проверки ввода для модели, марки, цены, пробега и лошадиных сил.

### Структура проекта

- `LabWork6.slnx` - файл решения Visual Studio;
- `LabWork6_Kunev_Anton` - основной проект Windows Forms;
- `MotorbikeDB.sql` - скрипт для создания базы данных;
- `FormShowMot` - главная форма со списком мотоциклов;
- `FormAddUPDMD` - форма добавления новой записи;
- `ModelEF` - классы для работы с базой данных через Entity Framework.

### Как запустить

1. Открыть SQL Server Management Studio или другой инструмент для MS SQL Server.
2. Выполнить скрипт `MotorbikeDB.sql`.
3. Открыть проект `LabWork6.slnx` в Visual Studio.
4. Проверить строку подключения в файле `LabWork6_Kunev_Anton/App.Config`.
   Сейчас используется:

```xml
data source=(localdb)\MSSQLLocalDB;initial catalog=MotorbikeDB;integrated security=True
```

5. Если Visual Studio попросит, восстановить NuGet-пакеты.
6. Запустить проект.

### Примечание

Папка `Pictures` создаётся автоматически рядом с исполняемым файлом программы. В неё копируются изображения, выбранные при добавлении мотоцикла.
